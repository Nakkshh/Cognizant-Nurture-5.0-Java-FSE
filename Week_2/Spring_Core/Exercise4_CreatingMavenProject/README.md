# Exercise 4 - Creating and Configuring a Maven Project

## Objective

Create a Maven project for the Library Management application and configure Spring Framework dependencies.

## Technologies

- Java 8
- Maven
- Spring Framework

## Dependencies Added

- Spring Context
- Spring AOP
- Spring WebMVC

## Maven Compiler Plugin

Configured to compile the project using Java 1.8.

## Verification

Run:

mvn validate

or

mvn compile

The build should complete successfully.
