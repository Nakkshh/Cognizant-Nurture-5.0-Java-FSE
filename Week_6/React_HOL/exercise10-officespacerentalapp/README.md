# Exercise 10 - Office Space Rental App

## Objective

Demonstrate JSX and Inline CSS in React.

## Features

- JSX
- React.createElement (via JSX)
- JavaScript Expressions
- Inline CSS
- Array.map()

## Technologies Used

- React
- JavaScript ES6

## Folder Structure

```
src
│── App.js
└── App.css
```

## Output

Displays multiple office spaces with image, rent and address.

Rent less than ₹60000 is displayed in Red.

Rent greater than or equal to ₹60000 is displayed in Green.
