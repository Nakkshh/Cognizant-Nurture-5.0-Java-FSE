# Exercise 12 - Ticket Booking App

## Objective

Demonstrate Conditional Rendering in React.

## Features

- Conditional Rendering
- Login Button
- Logout Button
- Guest View
- User View

## Technologies Used

- React
- JavaScript

## Folder Structure

```
src
│── LoginButton.js
│── LogoutButton.js
│── UserGreeting.js
│── GuestGreeting.js
│── Greeting.js
│── App.js
```

## Output

- Guest page shown before login.
- User page shown after login.
- Logout returns to guest page.
