# Cognizant Digital Nurture 5.0 – Java FSE (React)

This repository contains my week-wise hands-on exercises, assignments, and projects completed as part of the **Cognizant Digital Nurture 5.0 (DN 5.0) Deep Skilling Program** under the **Java Full Stack Engineer (React)** learning track.

The program covers core Java, Spring Framework, React, DevOps, Cloud fundamentals, Git, CI/CD, and software engineering best practices through practical hands-on exercises.
