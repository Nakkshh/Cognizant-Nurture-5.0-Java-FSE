# Week 5 - React Hands-On Labs



## Overview



This week focuses on learning the fundamentals of React by building multiple hands-on applications. The exercises cover creating React applications, working with components, props, state, lifecycle methods, styling, and modern React development practices.



## Learning Objectives



- Understand Single Page Applications (SPA)

- Learn React architecture and Virtual DOM

- Create and use Class and Functional Components

- Pass data using Props

- Manage Component State

- Understand React Component Lifecycle

- Apply CSS Modules and Inline Styling

- Build reusable React components



## Exercises Completed



### Exercise 1 - Create First React App



- Created the first React application using Create React App

- Displayed a welcome message

- Learned React project structure

- Executed the application using npm



### Exercise 2 - Student App Components



- Created multiple Class Components

- Implemented Home, About and Contact components

- Rendered multiple components inside App

- Understood component-based architecture



### Exercise 3 - Score Calculator App



- Created a Functional Component

- Passed values using Props

- Calculated average score

- Applied external CSS styling

- Rendered styled output



### Exercise 4 - Blog App Lifecycle



- Created a Blog application

- Implemented componentDidMount()

- Implemented componentDidCatch()

- Loaded data using Fetch API

- Displayed dynamic blog posts



### Exercise 5 - Styling React Components



- Applied CSS Modules

- Used className for component styling

- Applied conditional inline styling

- Styled cohort dashboard cards

- Displayed different colors based on cohort status
