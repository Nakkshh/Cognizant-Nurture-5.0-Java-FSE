# Exercise 4 - Blog App Lifecycle

## Objective

Build a React application to demonstrate React Component Lifecycle methods.

## Features

- Created a React application named **blogapp**
- Created a Post model
- Implemented a Posts class component
- Loaded posts using Fetch API
- Used componentDidMount() to fetch data
- Used componentDidCatch() for error handling
- Displayed blog posts dynamically

## Technologies Used

- React
- JavaScript (ES6)
- Fetch API
- Node.js
- npm

## Project Structure

```
blogapp/
├── src/
│   ├── App.js
│   ├── Post.js
│   ├── Posts.js
│   └── index.js
├── public/
├── package.json
└── README.md
```

## Output

The application fetches and displays blog posts from the JSONPlaceholder API.

## Learning Outcome

- Understood React Component Lifecycle
- Implemented componentDidMount()
- Implemented componentDidCatch()
- Worked with Fetch API
- Managed component state
