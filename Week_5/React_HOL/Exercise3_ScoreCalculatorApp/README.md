# Exercise 3 - Score Calculator App

## Objective

Build a React application that demonstrates the use of Functional Components, Props, and CSS styling by calculating and displaying a student's average score.

## Features

- Created a React application named **scorecalculatorapp**
- Implemented a Functional Component named **CalculateScore**
- Passed student details using React Props
- Calculated the student's average score
- Applied external CSS styling
- Rendered the component in the main application

## Technologies Used

- React
- JavaScript (ES6)
- JSX
- CSS
- Node.js
- npm

## Output

The application displays:

- Student Name
- School Name
- Total Marks
- Goal
- Average Score

with custom CSS styling.

## Learning Outcome

- Understood Functional Components in React
- Learned how to pass data using Props
- Applied external CSS to React components
- Practiced component rendering and reusable UI development
