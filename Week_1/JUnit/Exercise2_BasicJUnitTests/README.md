# Exercise 2 - Writing Basic JUnit Tests

## Objective

This exercise demonstrates how to write basic JUnit test cases for a Java class.

## Methods Tested

* Addition
* Subtraction
* Multiplication
* Division

## Technologies Used

* Java 17
* Maven
* JUnit 4.13.2

## Run

```bash
mvn test
```

## Expected Result

All test cases should pass successfully with BUILD SUCCESS.
