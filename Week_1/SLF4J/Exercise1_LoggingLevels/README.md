# Exercise 1 - Logging Error Messages and Warning Levels

## Objective

Demonstrate logging using the SLF4J API with Logback.

## Technologies Used

* Java 17
* Maven
* SLF4J
* Logback

## Execution

```bash
mvn compile
mvn exec:java -Dexec.mainClass="com.cognizant.LoggingExample"
```

## Expected Output

The application logs one ERROR message and one WARN message to the console.
