# Exercise 4 - Merge Conflict Resolution

## Objective

Learn how to identify and resolve merge conflicts in Git.

## Commands Used

```bash
git init
git branch feature
git checkout feature
git add .
git commit
git checkout main
git merge feature
git add .
git commit
git log --oneline --graph
```

## Learning Outcome

- Created two branches.
- Modified the same file in both branches.
- Triggered a merge conflict.
- Resolved the conflict manually.
- Completed the merge successfully.
