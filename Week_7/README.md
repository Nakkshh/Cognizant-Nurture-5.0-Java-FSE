# Week 7 - Git Hands-On Exercises

## Overview

This week focused on learning the fundamentals of Git and version control. The exercises covered Git configuration, repository initialization, file ignoring using `.gitignore`, branching and merging, merge conflict resolution, and working with remote GitHub repositories.

---

## Exercises Completed

### Exercise 1 - Git Configuration
- Installed and verified Git.
- Configured global username and email.
- Initialized a local Git repository.
- Verified Git configuration.

### Exercise 2 - Git Ignore
- Created a `.gitignore` file.
- Ignored unnecessary files using wildcard patterns.
- Verified ignored files using `git status`.

### Exercise 3 - Branching and Merging
- Created a new Git branch.
- Switched between branches.
- Committed changes in a feature branch.
- Merged the feature branch into the main branch.
- Verified commit history.

### Exercise 4 - Merge Conflict Resolution
- Created conflicting changes in two branches.
- Triggered a merge conflict.
- Resolved the conflict manually.
- Successfully completed the merge process.

### Exercise 5 - Working with Remote Repository
- Initialized a local Git repository.
- Created and committed project files.
- Added a remote GitHub repository.
- Verified remote configuration.
- Learned the process of pushing commits to GitHub.
