# Exercise 3 - Branching and Merging

## Objective

Learn Git branching and merging.

## Commands Used

```bash
git init
git branch feature
git checkout feature
git add .
git commit
git checkout main
git merge feature
git log --oneline --graph
```

## Learning Outcome

- Created a new branch.
- Switched between branches.
- Committed changes in the feature branch.
- Merged the feature branch into the main branch.
- Verified commit history.
