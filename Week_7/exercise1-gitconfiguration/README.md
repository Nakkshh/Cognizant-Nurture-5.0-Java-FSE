# Exercise 1 - Git Configuration

## Objective

Configure Git and initialize a local Git repository.

## Commands Executed

```bash
git config --global user.name "Omkar Mahesh"
git config --global user.email "your-email@example.com"

mkdir GitDemo
cd GitDemo

git init

git status
```

## Output

- Configured Git username and email.
- Initialized an empty Git repository.
- Verified repository status using `git status`.
