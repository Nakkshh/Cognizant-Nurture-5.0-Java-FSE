# Exercise 2 - Git Ignore

## Objective
Learn how to ignore unnecessary files using `.gitignore`.

## Steps Performed

1. Initialized a Git repository.
2. Created sample project files.
3. Added a `.gitignore` file.
4. Ignored `.txt` and `.log` files.
5. Verified ignored files using `git status`.
6. Added remaining files using `git add .`.
